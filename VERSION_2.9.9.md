# 版本 2.9.9 发布说明

## 发布信息
- **版本号**: 2.9.9
- **发布名称**: 筛选功能优化与编辑框滚动支持
- **发布日期**: 2025-10-22
- **构建时间**: 2025-10-22T20:45:00+08:00

## 概述
这是一个重要的用户体验优化版本，主要优化了所有页面的筛选功能交互，并完善了"本周进展/问题"编辑框的滚动支持，让长文本编辑更加便捷。

## ✨ 新增功能

### 筛选功能优化
1. **取消选择按钮**
   - 在所有页面的筛选下拉菜单中（除按KR筛选）新增"取消选择"按钮
   - 与"全部选择"按钮并排显示，方便一键清空所有选项
   - 当没有选中项时，按钮自动置灰，提供清晰的状态反馈

2. **优化的按钮布局**
   - 使用 flex 布局让两个按钮平均分配空间
   - "全部选择"使用主题紫色背景，视觉更突出
   - "取消选择"使用灰色边框样式，作为次要操作

### 编辑框滚动支持
1. **上下滚动功能**
   - "本周进展/问题"编辑框支持垂直滚动
   - 内容超过最大高度时自动显示滚动条
   - 编辑框高度根据内容自适应调整

2. **宽度固定优化**
   - 编辑框宽度固定为容器宽度
   - 完全禁止横向滚动
   - 长文本自动换行，不会超出容器

## 🔧 技术改进

### MultiSelectDropdown 组件优化
- 将单个切换按钮改为两个并排按钮
- "全部选择"按钮保持选中所有筛选结果的功能
- "取消选择"按钮支持一键清空所有已选项
- 添加禁用状态，当无选中项时按钮置灰

### RichTextInput 组件优化
1. **adjustHeight 函数增强**
   - 添加 `element.style.overflowX = 'hidden'` 禁止横向滚动
   - 动态控制 `overflowY` 属性（内容超过最大高度时显示滚动条）
   - 确保编辑框高度在 minHeight 和 maxHeight 之间

2. **内联样式优化**
   - 添加 `overflowX: 'hidden'` 确保宽度固定
   - 保留文本自动换行相关样式：
     - `wordWrap: 'break-word'`
     - `wordBreak: 'break-word'`
     - `overflowWrap: 'anywhere'`
     - `whiteSpace: 'pre-wrap'`

3. **CSS 样式完善**
   - 添加 `overflow-x: hidden !important` 强制禁用横向滚动
   - 确保所有子元素也遵循宽度限制

### 组件使用方式统一
- `ProjectDetailModal.tsx` - 使用 `minRows={4}` 和 `maxRows={12}` 控制高度
- `WeeklyMeetingProjectCard.tsx` - 使用 `minRows={4}` 和 `maxRows={10}` 控制高度
- `RichTextEditableCell.tsx` - 已设置 `minRows={8}` 和 `maxRows={25}`
- 移除所有固定 min-h 或 max-h 类名，使用组件属性统一控制

## 🎯 使用指南

### 筛选功能
1. 点击任意筛选下拉菜单
2. 可以看到"全部选择"和"取消选择"两个按钮
3. 点击"全部选择"会选中当前筛选结果中的所有选项
4. 点击"取消选择"会清空所有已选项
5. 当没有选中项时，"取消选择"按钮会自动置灰

### 编辑框滚动
1. 在"本周进展/问题"编辑框中输入大量文本（超过10行）
2. 编辑框会自动调整高度到最大值
3. 超过最大高度后，会显示垂直滚动条
4. 文本会自动换行，不会产生横向滚动
